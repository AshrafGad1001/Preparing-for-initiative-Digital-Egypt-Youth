## 🎨 Color Palette Used

| Color    | Hex       | Used For                    |
|----------|-----------|-----------------------------|
| Orange   | `#E87F24` | Buttons, accents, highlights |
| Dark Gray | `#454040` | Body text, paragraphs       |
| Light    | `#F5F5F5` | Section backgrounds, cards  |
| Navy     | `#003049` | Headings, navbar active     |
| Darkest  | `#091413` | Footer background           |

## 🚀 How to Use

1. **Create an `images/` folder** next to the HTML files
2. **Add your images** with the names listed above
3. **Open `index.html`** in your browser — done!

> ⚠️ If an image is missing, a green placeholder color appears automatically.
